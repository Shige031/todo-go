# TODO リスト - Go

## Description

Go の基本的な文法を学習するための TODO リストです。
レイヤードアーキテクチャをもう少し疎結合にした感じで作りました。
非常に簡素なのであしからず。。

## Get Started

```
docker-compose up -d
go mod tidy
go run main.go
```
